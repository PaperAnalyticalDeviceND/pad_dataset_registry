# Contributing

Thanks for your interest in contributing to the PAD Dataset Registry.

For adding a new dataset, see **[How to add a dataset](docs/how-to-add-a-dataset.md)** — the canonical step-by-step procedure, covering the required file list, the 8-column metadata schema, the Croissant generation step, local validation, the PR flow, and common gotchas. The same page is published on the catalog site at <https://padproject.info/pad_dataset_registry/how-to-add-a-dataset.html>.

For bug reports about the catalog generator, the Croissant generator, or the validator: please open an issue.

For broader questions about Paper Analytical Devices and the project: see <https://padproject.info/>.
